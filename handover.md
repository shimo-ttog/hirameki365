# プロジェクト引き継ぎ書：ひらめき365

## 1. プロジェクト概要
- **名称**: ひらめき365：大人の脳活・記憶力トレーニング
- **目的**: 大人世代が毎日達成感を得られる、目に優しく知的な脳トレWebアプリ。「サードプレイス」としての価値提供。
- **技術スタック**: Astro (v5.x), Node.js (ESM), Web Speech API, canvas-confetti, PWA (Service Worker)

## 2. 実装済みの主要機能 (2026-05-10 更新 - v0.1.9)
- [x] **7問構成の脳活スイート**: 昭和回想、平成回想、買い物計算、難読漢字、言葉パズル、ひらめき、記憶力、間違い探し。
- [x] **1問ずつのステップ進行UI**: 集中力を高める1画面1問形式。オートフォーカス＆Enterキー対応。
- [x] **アクセシビリティ**: 3段階文字サイズ変更、音声読み上げ（難読漢字は伏せ読み）、効果音（カスタム音源）。
- [x] **柔軟な正否判定**: カンマ、単位、曜日、別名（カッコ書き）を自動吸収。二重加算防止ロジック。
- [x] **カレンダー動的化**: 実行時の日付に基づき「今日」を表示。過去のスコア（N/7）と💮表示。未来制限。
- [x] **サードプレイス要素**: 豆知識、今日の格言（外部JSON管理）、SNS報告ボタン（LINE/X）。
- [x] **PWA化（ホーム画面に追加）**: オフライン対応、インストールバナー、アプリ風の全画面表示。
- [x] **GitHub・自動デプロイ**: さくらインターネットへの自動転送環境（rsync via GitHub Actions）。
