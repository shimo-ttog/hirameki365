# プロジェクト引き継ぎ書：ひらめき365

## 1. プロジェクト概要
- **名称**: ひらめき365：大人の脳活・記憶力トレーニング
- **目的**: 大人世代が毎日達成感を得られる、目に優しく知的な脳トレWebアプリ。「サードプレイス」としての価値提供。
- **技術スタック**: Astro (v5.x), Node.js (ESM), Web Speech API, canvas-confetti

## 2. 実装済みの主要機能 (2026-05-10 更新)
- [x] **7問構成の脳活スイート**: 昭和・平成回想（平成15年頃まで）、買い物計算、難読漢字、言葉パズル、ひらめき、記憶力、間違い探し。
- [x] **1問ずつのステップ進行UI**: 集中力を高める1画面1問形式。オートフォーカス＆Enterキー対応。
- [x] **アクセシビリティ**: 3段階文字サイズ変更、音声読み上げ（難読漢字は伏せ読み）。
- [x] **柔軟な正否判定**: カンマ、単位、曜日、別名（カッコ書き）を自動吸収。
- [x] **カレンダー動的化**: 実行時の日付に基づき「今日」を表示し、未来の問題へのアクセスを制限。
- [x] **達成感演出**: 紙吹雪、条件付き合格スタンプ、カレンダーへのスコア（N/7）＆💮表示。
- [x] **サードプレイス要素**: 豆知識、今日の格言（外部JSON管理）。
