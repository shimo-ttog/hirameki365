# 今日の格言：選定と追加のルール

「ひらめき365」がユーザーにとっての「サードプレイス（心安らぐ居場所）」であるために、以下のルールに基づいて格言を管理します。

## 1. コンテンツの基準
- **前向きであること**: 読んだ後に少しでも気持ちが明るくなる言葉を選びます。
- **説教臭くないこと**: 「〜すべき」という強制ではなく、「〜してみませんか」「〜ですね」という寄り添う姿勢を大切にします。
- **シニアに寄り添う**: 昭和の偉人、文化人、あるいは健康や知的好奇心を刺激する言葉を優先します。
- **短く簡潔に**: パッと見て3秒で心に入る長さを目安にします。

## 2. カテゴリー構成
格言は以下の4つの要素をバランスよく配合します：
1. **偉人の名言**: 歴史的な人物やスポーツ選手の力強い言葉。
2. **日常の癒やし**: 季節の移ろいや、笑顔の尊さを説く言葉。
3. **脳活の励まし**: 脳トレを頑張る姿勢を肯定し、応援する言葉。
4. **健康への気づき**: 散歩や食事など、健やかな生活を促す言葉。

## 3. 追加手順
1. `data/maxims.json` の配列に新しい文字列を追加します。
2. 格言は重複を避け、出典がある場合は可能な限り（）で記載します。
3. 保存後、`npm run augment`（または相当のスクリプト）を実行してコンテンツに反映させます。

## 4. 禁止事項
- 政治的・宗教的な偏りのある言葉。
- 誰かを批判したり、否定したりする言葉。
- 過度に難しい熟語や、解釈が分かれすぎる難解な言葉。
